### Factorial

```
RubyVM::InstructionSequence.compile_option = { tailcall_optimization: true, trace_instruction: false}

eval <<CODE
  def factorial(n, result=1)
    if n == 1
      result
    else
      factorial(n-1, n*result)
    end
  end
CODE

p factorial(30_000)
exit

```
